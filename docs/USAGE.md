# Usage Guide

## Running NetSleuth

### Windows
```bash
.\run.bat
```

### Linux/macOS
```bash
./run.sh
```

## Interactive Menu

1. **Analyze PCAP** - Load and analyze a PCAP file
2. **Display Devices** - Show discovered devices
3. **Export Devices (JSON)** - Export to JSON format
4. **Export Devices (CSV)** - Export to CSV format
5. **Exit** - Quit the application

## Example Workflow

1. Run NetSleuth
2. Select "Analyze PCAP"
3. Enter path to your PCAP file
4. View results with "Display Devices"
5. Export results as JSON or CSV
